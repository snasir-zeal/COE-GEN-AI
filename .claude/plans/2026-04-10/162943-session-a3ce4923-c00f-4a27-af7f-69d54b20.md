# Fix: Uniform alignment across all two-column slides

## Context
The presentation has inconsistent horizontal alignment across slides. Content starts at different x-positions, gaps between left/right columns vary (24px–56px), and column ratios differ (1fr/1fr vs 1fr/1.5fr vs 1fr/1.6fr). This makes the presentation feel visually uneven when scrolling through slides.

## Root cause
Three independent layout parameters vary across slides with no consistent system:
1. **Slide horizontal padding**: 72px, 48px, or 48px depending on slide class
2. **Column gap**: 56px (default `.slide-split`), 40px (inline override), or 24px (inline grid)
3. **Column ratio**: 1fr/1fr, 1fr/1.5fr, or 1fr/1.6fr

## Approach: Standardize horizontal padding and gap for all two-column slides

### Step 1 — Normalize horizontal padding
All slides should use the same **horizontal** padding so left-aligned content starts at the same x-position. The vertical padding can still differ (compact slides need less vertical space).

- Change `.slide` to `padding: 80px 60px`
- Change `.slide--compact` to `padding: 48px 60px`
- Change `.slide--semi-compact` to `padding: 60px 60px`

This gives every slide the same 60px left/right inset, while keeping vertical variation intact.

### Step 2 — Standardize the gap on `.slide-split`
Change the default `.slide-split` gap from 56px to **40px** and remove all inline `gap` overrides on slides 6, 7, 8.

### Step 3 — Convert inline grids (slides 10, 11) to use `.slide-split`
Slides 10 and 11 use ad-hoc inline `display:grid` instead of the `.slide-split` class. Convert them:
- **Slide 10** (The Reasoner): wrap content in `.slide-split` with a new modifier for wider right column: `.slide-split--wide-right` (`grid-template-columns: 1fr 1.4fr`)
- **Slide 11** (Production Output): same `.slide-split--wide-right`

This ensures they use the same gap (40px) and the same horizontal padding (60px).

### Step 4 — Remove `max-width: 1100px` from slides 10 and 11 inline grids
The `.slide-split` will naturally fill the available width within the standardized padding, just like every other two-column slide.
